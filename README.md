# AI Gemini Portfolio Project

This project demonstrates how I use **Google Gemini** across different stages of the data workflow:  
from extracting information from unstructured text, to analyzing datasets, building visualizations, creating an interactive dashboard, and developing a small AI-powered application for data cleaning.

The project is structured into four parts, each showing a different practical use case of Gemini for data work.

## 1️⃣ Omakanta Reviews — Extracting and Structuring Text Data

**Goal:**  
Show how Gemini can extract information from unstructured text (PDF screenshots) and convert it into a structured dataset.

**What I did:**
- Uploaded a PDF with user reviews from the Omakanta app  
- Used Gemini to extract dates, text, and star ratings  
- Used AI functions in Google Sheets to clean and structure the data  
- Produced a clean dataset ready for analysis

**Output:**  
- `Omakanta Reviews Dataset.xlsx`  
- `Omakanta_Extraction_Notes.md`

---

## 2️⃣ Data Analysis with Gemini in Google Sheets

**Goal:**  
Demonstrate how Gemini can assist with high-level data analysis and visualization.

**What I did:**
- Used a subset of Kela benefit data (2021–2024)
- Asked Gemini to calculate totals and summarize key metrics
- Asked analytical questions to generate narrative insights
- Created visualizations directly through Gemini in Sheets

**Output:**  
- `Build_with_AI_Data_Analysis_Report.docx`  
- `data_2021_2024.csv`

---

## 3️⃣ Interactive Dashboard with Gemini + BigQuery

**Goal:**  
Show how Gemini can help build a dashboard and explore data interactively.

**What I did:**
- Aggregated the large Kela dataset using SQL in BigQuery
- Uploaded the aggregated dataset to Gemini
- Generated charts, summaries, and dashboard components
- Created a small interactive dashboard for exploring benefit payments

**Output:**  
- `Gemini_Data_Visualization_Report_Kela.docx`  
- `Kela_Aggregated_Data_2021_2024.csv`  
- `SQL_used_in_BigQuery.sql`

---

## 4️⃣ CSV Cleaner App — AI-Powered Data Cleaning Tool

**Goal:**  
Create a simple AI application that automates repetitive data cleaning tasks.

**What I did:**
- Defined the workflow and requirements for a CSV cleaning tool
- Built the app in Google AI Studio
- Implemented automatic detection of duplicates, missing values, inconsistent formats, and outliers
- Exported a clean dataset ready for analysis

**Output:**  
- `CSV_Cleaner_App_Report_Oksana.docx`  
- `Workflow_Table_CSV_Cleaner_App.docx`  
- `app_link.txt`
